# 🤖 AI Chatbot using DialoGPT

## 📌 Description
This project is an AI chatbot built using Microsoft's DialoGPT model. It generates conversational responses and maintains context using Natural Language Processing (NLP).

## 💡 Features
- Context-aware conversation
- Pre-trained conversational AI model
- Interactive web interface using Gradio

## 🛠 Technologies Used
- Python
- HuggingFace Transformers
- Gradio

## 🚀 How to Run
1. Install dependencies:
   pip install transformers gradio torch

2. Run the program:
   python chatbot.py

## 📷 Output
This chatbot responds to user queries and maintains conversation context.
